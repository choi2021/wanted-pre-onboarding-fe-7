# 원티드 프리온보딩 사전과제


:: 1. 로그인 / 회원가입 페이지

로그인과 회원가입 두가지를 별도로 두지않고 하나의 페이지로 보이게 디자인했습니다. 
로그인과 회원가입은 동일하게 이메일과 비밀번호를 작성할 수 있는 input들과 제출할 수 있는 버튼이 있습니다.
이메일은 @를 반드시 포함해야하고, 비밀번호는 8자이상이 되어야 버튼이 활성화 될 수 있게 되어있습니다.
